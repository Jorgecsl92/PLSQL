## Descripción del proyecto

Este repositorio forma parte del proyecto final del curso **Desarrollador Oracle SQL-PL/SQL** impartido por **CAS Training**.  
Su objetivo es recopilar, organizar y documentar los ejercicios y prácticas de **PL/SQL**, creando un recurso útil para consulta futura y desarrollo profesional.

El contenido incluye:

- Procedimientos almacenados
- Funciones
- Cursores
- Excepciones
- Bucles y estructuras de control
- Ejercicios aplicados a tablas EMP, DEPT, DOCTOR, HOSPITAL, etc.

---

## Contenido del repositorio

Este repositorio **PLSQL** incluye:

- Declaración de variables  
- Bucles (`FOR`, `WHILE`, `LOOP`)  
- Cursores explícitos e implícitos  
- Manejo de excepciones  
- Procedimientos y funciones  
- Ejemplos prácticos comentados

---

## Objetivos del proyecto

- Aplicar PL/SQL a casos prácticos reales.
- Automatizar operaciones sobre bases de datos.
- Comprender el uso de cursores, excepciones y procedimientos.
